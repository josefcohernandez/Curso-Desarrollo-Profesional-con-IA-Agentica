# Curso de Desarrollo Profesional con IA AgГ©ntica

> **Domina el oficio de desarrollar software con agentes de cГіdigo**

## Acerca de este curso

Este curso enseГұa las habilidades, el pensamiento crГӯtico y las metodologГӯas que necesitas para trabajar de forma profesional y efectiva con agentes de cГіdigo IA. No se centra en una herramienta especГӯfica вҖ” los conceptos son transferibles entre Claude Code, Cursor, Copilot, Windsurf, Cline o cualquier herramienta de coding con IA.

Mientras que un curso de herramienta enseГұa *quГ© botones pulsar*, este curso enseГұa **cГіmo pensar**: escribir prompts efectivos, detectar errores del agente, revisar cГіdigo generado por IA, debuggear de forma sistemГЎtica, y adoptar IA en equipos de desarrollo.

### PГәblico objetivo

- Desarrolladores que ya usan Claude Code u otra herramienta de coding con IA y quieren ser mГЎs efectivos
- Usuarios de Cursor, Copilot, Windsurf o Cline que quieren mejorar su forma de trabajar
- Equipos evaluando quГ© herramienta adoptar y cГіmo integrarla en su workflow
- Tech leads y managers que necesitan entender la disciplina sin dominar una herramienta especГӯfica
- Cualquier desarrollador que quiera prepararse para el futuro del desarrollo de software

### Requisitos previos

- Experiencia en programaciГіn (cualquier lenguaje)
- Haber usado al menos una herramienta de coding con IA (no necesitas ser experto)
- Familiaridad con conceptos bГЎsicos de desarrollo: tests, control de versiones, code review

**Recomendado**: si usas Claude Code, haber completado los mГіdulos M01-M06 del [Curso de Claude Code](https://github.com/josefcohernandez/claude-code-course) o tener experiencia equivalente.

Sigue los bloques en orden (A вҶ’ B вҶ’ C вҶ’ D вҶ’ E вҶ’ Proyecto). Cada mГіdulo construye sobre los anteriores. Dedica al menos 1 semana al Bloque A вҖ” es la base de todo lo demГЎs. El Bloque E (Experto) es opcional si no necesitas nivel avanzado.
